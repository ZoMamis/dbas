Software Stack¶

Briefly overview of the software stack.

Server (backend)¶

  • Python 3 (programming language)
  • Pyramid (web framework)
  • ZODB (database)
  • SQLAlchemy (database)
  • Colander (data schema)
  • Sphinx (documentation)
  • Chameleon (HTML/XML template engine for Python)

Client (frontend)¶

  • JavaScript (programming language)
  • HTML5 (text markup)
  • Bootstrap (CSS preprocessor)
  • JQuery (javascript helper library)
  • Jasmine (unit tests)
  • Splinter (acceptance and front end tests)
